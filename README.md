# 2D Robot Kinametics

## Requirements
- numpy
- OpenCV

## Run
```
python3 fabrik_basic.py
```
Forward And Backward Reaching Inverse Kinematics on a simple 5 link kinematic chain.
Click to place the target position.

---

```
python3 fabrik_cspace.py
```
Forward And Backward Reaching Inverse Kinematics on a 2 link kinematic chain in an environment with colliders.
Click and drag boxes or target position.
